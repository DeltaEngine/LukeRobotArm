Controller

Website to control the robot arm, usually launched from a phone (Android or iOS), but can be also used from your laptop or PC (Windows, Mac, Linux) to control Luke.

LukeRobotArm.com

If you just want to use your Luke Robot Arm, you don't have to download anything, just go to [LukeRobotArm.com](https://lukerobotarm.com) and follow the instructions.

1. Connect to the Luke- wifi and open http://Luke or http://192.168.100.1, there you can enter your local wifi credentials so the robot can talk to your phone. 2. Go back to your wifi and connect to Luke- that should appear on https://LukeRobotArm.com 3. Once connected you can control the robot with your phone or laptop using voice, camera, controls and machine learning (AI).

Server

Luke Robot Arm Server on TTGO T-Display ESP32 Arduino Project. This is the heart of the robot arm, it is the server that controls the robot arm and communicates with the phone app. It runs on the esp32 ttgo t-display arduino device that is built in and you control the robot with your phone or laptop using voice, camera, controls and machine learning (AI) via the web app. Or use one of the programming examples (Python/C#/..). It is pretty dumb, low power and can stay on all the time. It just follows the instructions you give it, all the fancy stuff happens on this website.
